\section{EDT Local Storage}
\label{sec:els}
This extension provides a dedicated portion of fixed-size pre-allocated
memory made available to each EDT for its private use, analogous to
TLS (Thread Local Storage). They are typically used by user-level
libraries and high-level language translators to maintain per-EDT state
in a uniform manner.

The extension proposes the use of the following API:
